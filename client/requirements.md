seedrandom | Deterministic RNG for game seeds

## Notes
Game logic is primarily Canvas-based but state is managed via React refs and state for UI.
API endpoints at /api/scores for submitting high scores.
Font requirements: Google Fonts 'Noto Sans SC' (for Chinese text) and 'Orbitron' (for futuristic numbers/UI).
